# Hands-on-Machine-Learning

🚀 **Hands-on Machine Learning with Python**  
A repository dedicated to exploring the fundamentals of **Machine Learning**, **Deep Learning**, and other advanced techniques through hands-on projects. This repo covers everything from basic machine learning algorithms to neural networks using **TensorFlow**, **Keras**, **scikit-learn**, and more! Based on the great book "Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow" by Aurelien Geron

---

## 📌 Features

- 🧠 **Learn by Doing:** A practical approach to understanding ML algorithms through real-world projects. I want to learn by doing


---

## 📖 Chapters

### Chapter 1: **Housing Price Estimation**  
This project uses a **Machine Learning model** to predict housing prices based on features like square footage, number of bedrooms, location, etc. It includes everything from data cleaning, model selection, training, evaluation, and deployment. A great starting point for beginners!

---



## ⚙️ Requirements

To get started with this repo, ensure you have the following installed:

- Python 3.x
- TensorFlow
- Keras
- scikit-learn
- pandas
- numpy
- scipy
- matplotlib

You can install all the required dependencies using:

```bash
pip install -r requirements.txt
```
